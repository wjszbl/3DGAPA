# GAN-Based GA for 3D Bin Packing

This project implements a GAN-based GA for the 3D bin packing problem.

## Requirements

- Python 3.8 or higher
- PyTorch 1.8 or higher
- numpy

## Usage

Run `python main.py` to start the algorithm.
